JavaScript

Was ist JavaScript?

JavaScript ist eine Skriptsprache, die zum Erstellen interaktiver Web-Seiten verwendet wird. Sie kann verwendet werden, um Animationen, Eingabefelder, Schaltflächen und andere Elemente zu erstellen, die Benutzerinteraktion ermöglichen.

Wie funktioniert JavaScript?

JavaScript wird in der Browser-Engine eines Web-Browsers ausgeführt. Wenn eine Web-Seite geladen wird, wird der JavaScript-Code in der Seite vom Browser ausgeführt.

JavaScript-Code kann in der HTML-Seite selbst eingebettet oder in einer separaten Datei gespeichert werden.

Was kann man mit JavaScript machen?

Mit JavaScript kann man Folgendes tun:

  • Animationen erstellen
  • Eingabefelder erstellen
  • Schaltflächen erstellen
  • Andere Elemente erstellen, die Benutzerinteraktion ermöglichen
  • Web-Seiten personalisieren
  • Web-Seiten interaktiver machen
